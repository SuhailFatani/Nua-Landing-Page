How to Use

Option A — Static BEM Footer

<!-- 1. Link CSS --> <link rel="stylesheet" href="Design system/Foundations/footer/footer.css"> <!-- 2. HTML structure --> <footer class="nua-footer"> <div class="nua-footer__inner"> <div class="nua-footer__content"> <div class="nua-footer__links-row"> <div class="nua-footer__brand"> <a href="index.html" class="nua-footer__logo">...svg...</a> <div class="nua-footer__social">...icons...</div> </div> <div class="nua-footer__nav"> <div class="nua-footer__nav-col"> <span class="nua-footer__nav-label">Links</span> <a href="company.html" class="nua-footer__nav-link">Company</a> </div> </div> </div> <div class="nua-footer__bottom"> <div class="nua-footer__badges">...ISO + Gartner...</div> <div class="nua-footer__contact">...email + copyright...</div> </div> </div> <div class="nua-footer__watermark">...svg...</div> <div class="nua-footer__fade"></div> </div> </footer>

Option B — Variant 1: Lottie Strip

<!-- Mount point --> <div id="cta-footer"></div> <!-- Script (before </body>) --> <script src="Design system/Components/Footer/Cta footer.js"></script>

Option C — Variant 2: Orb Edition

<!-- Mount point (different id) --> <div id="cta-footer-2"></div> <!-- Script (before </body>) --> <script src="Design system/Components/Footer/Cta footer 2.js"></script> <!-- Optional: same data-* API as Variant 1 --> <div id="cta-footer-2" data-heading="Custom Heading" data-subheading="Custom subheading text" data-cta-text="Get Started" data-cta-link="book-a-demo.html" data-show-cta="true" data-email="hello@example.com" data-copyright="© 2025 Company."> </div> <!-- Footer only (hide CTA) --> <div id="cta-footer-2" data-show-cta="false"></div>

Data Attributes

Attribute Default Description
data-heading "Get Ahead Of Threats" CTA heading text
data-subheading Long default text CTA subtitle text
data-cta-text "Book a demo" CTA button label
data-cta-link "book-a-demo.html" CTA button href
data-show-cta "true" Show/hide the CTA card ("false" = footer only)
data-email "info@nuasecurity.com" Contact email in footer
data-copyright "© 2025 NUA USA." Copyright text
